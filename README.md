## Description

오프라인 매장에서 발급되는 종이 쿠폰은 훼손되어 버려지거나 분실되는 경우가 많습니다.  
또한, 매장마다 쿠폰을 따로 관리하기 때문에 이를 관리해야 하는 고객이나 가게 입장에서 불편함이 있습니다.  
디지쿠는 그러한 문제점을 해결하고자 통합된 쿠폰관리 서비스를 제공합니다.
